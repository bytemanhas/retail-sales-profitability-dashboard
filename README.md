# 📊 Retail Sales Performance Dashboard (Power BI)

## 📌 Project Overview
This project presents a Retail Sales Performance Dashboard built using Power BI to analyze sales trends, product performance, and regional distribution. The dashboard provides key insights to support data-driven business decisions.

---

## 🎯 Objective
The objective of this project is to analyze retail sales data to:
- Identify top-performing products and categories
- Analyze sales trends over time
- Understand regional performance
- Provide actionable business insights

---

## 🛠️ Tools & Technologies
- Power BI
- Microsoft Excel (Data Preparation)
- DAX (Data Analysis Expressions)

---

## 📂 Dataset Description
The dataset contains retail sales transactions with the following fields:
- Order ID
- Product
- Category
- Sales
- Quantity
- Region
- Order Date

---

## 📊 Key Features of Dashboard
- KPI Cards:
  - Total Sales
  - Total Orders
  - Average Sales

- Visualizations:
  - Sales by Category (Bar Chart)
  - Sales by Region (Bar Chart)
  - Monthly Sales Trend (Line Chart)
  - Top Selling Products

- Interactive Filters:
  - Region
  - Category
  - Date

---

## 🔍 Key Insights
- Certain product categories generate significantly higher revenue
- Some regions contribute more to overall sales performance
- Sales trends show variation across different months indicating seasonal demand
- Top-selling products drive a major portion of total revenue

---

## 💡 Business Recommendations
- Focus marketing efforts on top-performing products
- Increase inventory for high-demand categories
- Target underperforming regions with promotional strategies
- Use seasonal trends to plan inventory and campaigns
